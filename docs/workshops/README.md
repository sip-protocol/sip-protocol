# SIP Protocol Developer Workshop Materials

Workshop resources for teaching developers to integrate privacy into their dApps.

## Workshop Overview

**Duration:** 90 minutes
**Target Audience:** Solana developers, dApp builders
**Prerequisite:** Basic Solana/Web3 knowledge

## For Workshop Presenters

### Setup Requirements

Ensure attendees have:
- Node.js 18+
- pnpm or npm
- A code editor
- Basic CLI knowledge

### Before the Workshop

1. Clone sample code repo
2. Install dependencies
3. Set up Helius API key (free tier works)
4. Test the starter template runs

### Materials Checklist

- [ ] Projector/screen setup
- [ ] Workshop slide deck loaded
- [ ] Sample repo cloned and tested
- [ ] Helius API keys distributed or attendees have own
- [ ] Discord/support channel ready for questions
