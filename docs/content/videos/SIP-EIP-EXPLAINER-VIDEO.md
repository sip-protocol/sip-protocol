# SIP-EIP Explainer Video Production Guide

**Video Title:** "SIP: The Privacy Standard for Web3 â€” Explained in 3 Minutes"
**Duration:** 3:00 - 3:30
**Style:** Motion graphics with voiceover
**Target:** General Web3 audience (developers + users)

---

## 1. Video Overview

### 1.1 Key Messages

1. Web3 privacy is currently broken (all-or-nothing)
2. SIP provides privacy + compliance through three primitives
3. It's easy to integrate (one toggle, 50 lines of code)
4. It's becoming a standard (SIP-EIP)

### 1.2 Tone

- **Accessible**: No jargon without explanation
- **Visual**: Show, don't just tell
- **Confident**: This is the solution, not "might be"
- **Inviting**: Join us in building this

### 1.3 Technical Requirements

- Resolution: 1920x1080 (16:9)
- Frame rate: 30fps
- Audio: Professional voiceover + subtle background music
- Captions: Burned-in subtitles for accessibility
- Export: MP4 (H.264) + WebM for web

## 4. Animation Specifications

### 4.1 Color Palette

| Element | Hex Code | Usage |
|---------|----------|-------|
| Primary | #8B5CF6 | SIP brand, highlights |
| Secondary | #06B6D4 | Accents, links |
| Success | #10B981 | Checkmarks, valid |
| Warning | #F59E0B | Caution, attention |
| Error | #EF4444 | X marks, problems |
| Background | #0F172A | Dark mode background |
| Text | #F8FAFC | Primary text |
| Muted | #94A3B8 | Secondary text |

### 4.2 Typography

| Element | Font | Weight | Size |
|---------|------|--------|------|
| Headings | Inter | Bold | 48px |
| Body | Inter | Regular | 24px |
| Code | JetBrains Mono | Regular | 20px |
| Labels | Inter | Medium | 18px |

### 4.3 Motion Guidelines

- **Easing:** Ease-out for entrances, ease-in-out for transitions
- **Duration:** 300ms for micro-interactions, 600ms for scene transitions
- **Stagger:** 100ms delay between sequential elements
- **Scale:** Elements enter at 0.9 scale, animate to 1.0

### 4.4 Icon Style

- Line weight: 2px
- Corner radius: 4px
- Style: Rounded, friendly
- Source: Lucide icons or custom

## 5. Audio Specifications

### 5.1 Voiceover

- **Voice:** Professional, warm, confident (not robotic)
- **Pace:** 150-160 words per minute
- **Tone:** Conversational but authoritative
- **Gender:** Neutral preference, focus on clarity

### 5.2 Music

- **Style:** Subtle electronic, ambient
- **Mood:** Optimistic, forward-looking
- **Volume:** -20dB under voiceover
- **License:** Royalty-free or licensed for commercial use

### 5.3 Sound Effects

| Moment | Sound | Volume |
|--------|-------|--------|
| Logo reveal | Subtle whoosh | -15dB |
| Checkmarks | Soft click | -18dB |
| Transitions | Gentle swoosh | -20dB |
| End card | Subtle flourish | -15dB |
