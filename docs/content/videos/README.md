## Production Pipeline

1. **Script Review**: Review and approve script content
2. **Storyboard**: Create visual storyboard from script
3. **Voice Recording**: Record voiceover
4. **Animation**: Create animations and visuals
5. **Editing**: Combine VO + visuals + music
6. **Review**: Internal review and feedback
7. **Publish**: Upload to YouTube, create shorts

## Video Specifications

### Long-Form (YouTube)
- Resolution: 1920x1080 (1080p)
- Frame rate: 30fps
- Format: MP4 (H.264)
- Aspect ratio: 16:9
- Max length: 5 minutes

### Short-Form (TikTok/Shorts/Reels)
- Resolution: 1080x1920 (vertical)
- Frame rate: 30fps
- Format: MP4
- Aspect ratio: 9:16
- Max length: 60 seconds

## Style Guide

### Visual Style
- Modern, clean aesthetic
- Dark mode preference (matches crypto audience)
- SIP Protocol brand colors
- Minimal text on screen
- Abstract technical visualizations

### Animation Style
- 2D motion graphics
- Smooth transitions
- No stock footage
- Custom illustrations preferred

### Typography
- Sans-serif fonts (Inter, Geist)
- Large, readable text
- Max 5-7 words per text overlay

### Music
- Electronic / ambient
- Non-distracting
- Royalty-free (Artlist, Epidemic Sound)
