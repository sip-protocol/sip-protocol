## 1. Securities Law Analysis (USA)

### 1.1 Howey Test Assessment

The Howey Test determines if an instrument is a "security" under US law.

| Prong | Question | SIP Analysis | Risk Level |
|-------|----------|--------------|------------|
| **Investment of Money** | Did purchasers invest money or value? | Yes â€” tokens acquired via purchase/contribution | Present |
| **Common Enterprise** | Is there horizontal or vertical commonality? | Partial â€” decentralized protocol, no single enterprise | Medium |
| **Expectation of Profits** | Do purchasers expect profits? | Mitigated â€” utility-first messaging, no profit promises | Low |
| **Efforts of Others** | Are profits derived from promoter efforts? | Mitigated â€” progressive decentralization, utility focus | Low |

### 1.2 Utility Token Characteristics

**Checklist â€” Document these characteristics:**

- [ ] Token provides genuine utility (governance voting, fee discounts)
- [ ] Token is functional at launch (not promise of future utility)
- [ ] Primary purpose is consumption, not investment
- [ ] Marketing emphasizes utility, not appreciation
- [ ] No guaranteed returns or profit sharing
- [ ] Value tied to network usage, not team efforts
- [ ] Decentralized governance reduces reliance on promoters

### 1.3 Risk Mitigations

| Risk | Mitigation Strategy |
|------|---------------------|
| Investment expectation | Utility-first messaging, no APY promises |
| Common enterprise | Progressive decentralization, DAO structure |
| Reliance on others | Community governance, open-source code |
| Marketing claims | Legal review of all public communications |

## 4. AML/KYC Requirements

### 4.1 KYC Tiers

| Tier | Transaction Limit | Requirements |
|------|-------------------|--------------|
| **Tier 0** | <$1,000 | No KYC (geo-block high-risk) |
| **Tier 1** | $1,000-$10,000 | Email, wallet verification |
| **Tier 2** | $10,000-$100,000 | ID verification (passport/DL) |
| **Tier 3** | >$100,000 | Full KYC + source of funds |

### 4.2 AML Compliance Checklist

- [ ] AML policy document
- [ ] KYC provider selected (Jumio, Onfido, etc.)
- [ ] Transaction monitoring system
- [ ] Sanctions screening (OFAC, UN, EU lists)
- [ ] Suspicious activity reporting procedures
- [ ] Record retention policy (5+ years)
- [ ] Compliance officer designated

### 4.3 FATF Travel Rule

For transfers >$1,000 (or local threshold):
- [ ] Collect originator information
- [ ] Collect beneficiary information
- [ ] Share with counterparty VASP
- [ ] Record-keeping compliance
